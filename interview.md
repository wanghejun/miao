# HTML
*　HTML语义化
```
1.方便开发与维护
2.其它设备也好解析，对视障用户更友好
3.有利于seo，搜索引擎根据标签来确定上下文和各个关键词来渲染权重

4.语义化标签:
header    头部
footer    尾部
nav       导航区，超连接区域
main      文档主要内容
section   一块区域
aside     侧边栏或插入内容

```
* !DOCTYPE html 头的意义

```

1.告诉浏览器html的版本
2.它必须处于文件的顶部，可以不区分大小写
3.浏览器通过它来判别使用什么渲染模式(标准模式，兼容模式)
```
